# sarcasm-detection_dsci6004_nlp
This project detects sarcasm using BiLSTM, DistilBERT, and RoBERTa models across multiple datasets.

Files:
Code: Scripts for training and evaluating models.
Datasets: Sarcasm Headlines, Sarcasm on Reddit, Hugging Face Combined Dataset.
Slides: Project summary and results.
Report: Detailed methodology and findings.
Documentation: Setup and usage instructions.

Usage:
Install dependencies:
pip install -r requirements.txt
Preprocess and train models using the scripts.
Evaluate results with the provided metrics.
